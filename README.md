# PUBLICO
Todos los proyectos que comparto con la comunidad.

En este repositorio podrás encontrar todo el código realizado en Java que _comparto_ contigo y con el resto de gente interesada en la programación en Java. Aquí encontrarás librerías desarrolladas por mí para mis proyectos, ejemplos de código que creo son relevantes para los programdores del mundo Java, así como artículos técnicos y su código asociado de mi web personal en [*Forastero en Tierra Extraña*](https://juanfranciscoruiz.nom.es/)

De entre los proyectos existentes en este repositorio puedo destacar el proyecto [*`ansiterm`*](ansiterm/README.md), que es una librería _modesta_ para poder manipular de forma adecuada la terminal mediante Java, más allá de lo que proporciona la IPA (Interfaz Pública de Aplicaciones) del lenguaje, pero sin llegar a los extremos de excelentes librerías para la gestión de TUIs (Terminal User Interfaces, o interfaces de terminal de usuario) como son *Lanterna* o *JAnsi*, por indicar las más conocidas.

Cualquier duda, crítica o mejora, podéis hacer pull-request o creer una issue en los apartados correspondientes.

Muchas gracias por haberle echado un vistazo a este humilde repositorio.
